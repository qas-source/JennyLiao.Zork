Here are the instructions for this game.
You type in a command or a verb/action to do something. For example, go north, go south, and look around.

Once typing in your action, click the button beside it. Once that is done, a print-out of what happened will appear bellow the text box. Use these actions to navigate within the tower. (Note that some actions may only work on specific places of the game and that some actions may only be available once another has been completed)

Here is a list of basic actions that can be used

look/look around

go north, go south, go east, go west. (You can also use n, w, e, and s for these)

observe ____ (can only be used on some floors)

open ____ (used to open unlocked doors, laptops, not books)

move ____

take ____ (can be used depending if it is takeable)

read _____

open door with _____ (if the door needs something to be opened)

enter password (to enter a password somewhere)

ask "_____" to ______ (to npcs in the game if there are)

give _____ (used when you can give something)

press ____ and press next _____ (use if needed)

These are the basic commands, with these, you should be able to clear the tower! Note that these are not all of them, and some of these commands can only be used in specific situations

If you are stuck, enter in "give hint" or "hint" might give you a hint.

Remember that everything should be lowercase unless it is a password or anything that says you have to enter something in capital letters.

Good luck getting to the top of the tower!
